본 사이트에 대한 문의사항이 있으시면 아래 방법을 통해 연락해 주세요.

## 연락 방법

### GitHub Issues
사이트 관련 문의, 오류 신고, 건의사항은 GitHub Issues를 통해 남겨주세요.

- [GitHub Issues 바로가기](https://github.com/jrkim3888/jrkim3888.github.io/issues)

### 댓글
각 게시물 하단의 댓글 기능(Giscus)을 통해서도 소통할 수 있습니다. 댓글 작성에는 GitHub 계정이 필요합니다.

## 문의 범위

- 사이트 이용 관련 문의
- 콘텐츠(소설) 관련 피드백
- 저작권 관련 문의
- 개인정보 관련 문의
- 광고 관련 문의
- 기타 건의사항

문의해 주시면 확인 후 가능한 빠르게 답변드리겠습니다. 감사합니다.
